# CRUD_Petshop
Tema escolhido: Sistema de Cadastro de Pets

O sistema foi desenvolvido para gerenciar os pets atendidos em um petshop.
Através de um menu no terminal, é possível cadastrar, listar, consultar,
atualizar e remover pets.

O CRUD foi aplicado da seguinte forma:
- Create: cadastro de novos pets
- Read: listagem geral e consulta por ID
- Update: atualização das informações do pet
- Delete: exclusão de pets do sistema

Os dados são armazenados em uma lista de dicionários e salvos
em um arquivo JSON para manter persistência.
